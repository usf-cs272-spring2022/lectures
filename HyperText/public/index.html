According to Wikipedia, the acronym TTHW stands for "time to hello world" and measures the ease of use of a programming language. Neat!

Java Example

Here is a simple Hello World program in Java: 

      public class Hello {
        public static void main(String[] args) {
          System.out.println("Hello world!");
        }
      }

It takes 3 source-lines-of-code (SLOC) to write this program.
